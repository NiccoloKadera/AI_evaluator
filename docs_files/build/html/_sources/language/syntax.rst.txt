ABML Syntax
===========

This page documents the syntax of the ABML language and its translation to Python.

.. list-table:: ABML Syntax Table
   :widths: 30 30 40
   :header-rows: 1

   * - Input
     - Output
     - Description

   * - ``(a > b) -> c``
     - ``if a > b: c``
     - This is the If statement.
   * - ``a = b``
     - ``a = b``
     - Assignment a = b -> a = b
